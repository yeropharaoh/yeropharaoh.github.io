Simple Packing-list Web app

A 'to-do-list' type exercise where the user can create an account, log in, select the type of vacation they are planning on taking, and can create a packing list which is added to their profile.
Uses: postgreSQL, bcrypt, jQuery, Bootstrap, pug, CSS3.
