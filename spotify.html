Playlist generator and beat-making music web app

Using Spotify's and Microsoft's Emotion API, a playlist can be generated based on the facial emotion portayed in an image.

*due to Spotify's recent API changes, the playlists can not be generated. Matching Youtube videos are used instead.

Uses: Rapid API, Emotion API, Spotify API, Web audio API, javascript, materialize, pug, CSS3


‹|
|›

The second feature lets you make music by playing/selecting loops that I created in Ableton Live.

The loops can either be clicked on or triggered by your keyboard! Some javascript and the Web audio API were used to achieve this.
