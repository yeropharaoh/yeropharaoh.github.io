Playlist generator and beat-making music app

As my final project for the NYCDA full-stack intensive, I used Microsoft's Emotion API to generate a YouTube video based on the facial emotion portayed in an uploaded image.

*due to Spotify's recent API changes, the code was changed to match YouTube videos instead.

Uses: Rapid API, Emotion API, Spotify API, Web audio API, JavaScript, Materialize, Pug, Css


‹|
|›

The second feature lets you make music by playing/selecting loops that I created in Ableton Live.

The loops can either be clicked on or triggered by your keyboard! Some JavaScript and the Web audio API were used to achieve this.
